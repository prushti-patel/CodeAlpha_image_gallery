# Image Gallery - CodeAlpha Internship
A beautiful, responsive, and interactive image gallery built using HTML, CSS, and JavaScript. This project was developed as part of the CodeAlpha internship program to showcase dynamic content filtering and responsive web design.

## Features
Dynamic Image Filtering: Easily filter images by categories like 'Nature' and 'City'.

Responsive Design: Optimized layout that works seamlessly across desktops, tablets, and mobile devices.

Interactive UI: Clean and modern interface for an enhanced user experience.

Easy Maintenance: Simple structure to add or remove images effortlessly.
